有渔 UI 库

用于公司项目的 UI 库

概述

项目所用到的样式中，大部分为 Boostrap，一部分为自己所写的公共样式，还有一小部分是其他第三方库和特定页面的。为了符合设计风格，对 Bootstrap 和其他第三方库的部分样式进行了重写。

本文档的主要内容为自己所写的公共样式。因此，导航栏菜单中没有关于第三方库的页面导向。Bootstrap 重写后的样子请看效果预览，具体用法请查阅官方文档。

用法

<link rel="stylesheet" href="./application.css">

<!-- 自定义过的 Bootstrap -->
<link rel="stylesheet" href="./bootstrap-custom.css">
