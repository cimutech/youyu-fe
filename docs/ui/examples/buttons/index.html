按钮

根据 Bootstrap 的按钮进行扩展而来。

目录

  • 种类
  • 大小

种类

只有两种无语义的按钮：红色的主按钮和蓝色的副按钮。

 
<!-- HTML -->

<button type="button" class="btn btn-red">主按钮</button>
<button type="button" class="btn btn-blue">副按钮</button>

其中，红色按钮的颜色为 #ED6553，蓝色按钮的颜色为 #7DC5FF。

大小

<!-- HTML -->

<div>
  <button class="btn btn-red btn-lg" type="button">大按钮</button>
  <button class="btn btn-blue btn-lg" type="button">大按钮</button>
</div>
<div>
  <button class="btn btn-red" type="button">普通按钮</button>
  <button class="btn btn-blue" type="button">普通按钮</button>
</div>
<div>
  <button class="btn btn-red btn-xs" type="button">小按钮</button>
  <button class="btn btn-blue btn-xs" type="button">小按钮</button>
</div>

各参数如下：

种类视觉高度字体大小字体颜色补白圆角大小
大按钮50px18px#FFFFFF纵：16px
横：20px		6px
普通按钮36px16px纵：10px
横：20px
小按钮28px14px纵：7px
横：20px
