文字

字号

项目里所用到的文字大小基本为四种：

  • 18px——大的标题；
  • 16px——小的标题；
  • 14px——内容的正文；
  • 12px——附加信息等。

这里所描述的文字大小为普通文本中所使用的，标题类的参照这里。

在 CSS 中定义了对应这四种文字大小的类供开发时使用。

大的文字18px
中等文字16px
小的文字14px
更小的字12px
<!-- HTML -->

<!-- 18px -->
<span class="Font--large">大的文字</span>
<!-- 16px -->
<span class="Font--medium">中等文字</span>
<!-- 14px -->
<span class="Font--small">小的文字</span>
<!-- 12px -->
<span class="Font--exSmall">更小的字</span>

标题

用于各种需要标题的场景。

此系列 HTML 标签具有「标题」的语义，所以只能在表示「标题」的地方使用，不要用在普通文本中！

如果是需要用到相同的文字大小，到这里；如果是同时想用同样的文字大小和颜色，就继续往下看。

h1. 标题

 

h2. 标题

 

h3. 标题

 

h4. 标题

 
<!-- HTML -->

<h1>h1. 标题</h1>
<h2>h2. 标题</h2>
<h3>h3. 标题</h3>
<h4>h4. 标题</h4>

每个级别的标题有各自的大小、颜色、高度等：

级别字体大小行高颜色边距
h118px1.5 倍#4C4C4C上：21px
下：10.5px
h216px1.5 倍#4C4C4C上：21px
下：10.5px
h314px1.5 倍#666666上：21px
下：10.5px
h412px1.5 倍#999999上：10.5px
下：10.5px

因为有些场景是非标题类的文本需要与标题类标签有相同的大小和颜色，所以提供了四个 Sass mixin。

// Sass

div {
  @include h1;
}

p {
  @include h2;
}

span {
  @include h3;
}

code {
  @include h4;
}

标点符号

一般情况下是用不到标点符号的处理的，但是在解析文字处理软件导出的文档（例如 Office Word 文档）时有可能会需要。

因为浏览器对 CSS 的兼容问题，为了保证在任何浏览器中都能够正常显示，需要调用脚本代码 YouYu.punctuate() 来确保。更多信息请看 YouYu.punctuate()。

着重号

有渔是最好的在线教育网站
<!-- HTML -->

<span class="punc-emphasis">有渔是最好的在线教育网站</span>
// JavaScript

// 兼容各浏览器
YouYu.punctuate($(".punc-emphasis"));

有渔大法好！

<span class="punc-guillemet">有渔大法好！</span>
// JavaScript

// 兼容各浏览器
YouYu.punctuate($(".punc-guillemet"));
