技术栈

从传统到追赶潮流的转变之中，项目中所使用的前端技术也在发生着变化。目前项目中所使用的核心技术如下：

  • jQuery
  • Backbone
  • Bootstrap

为了弥补 Backbone 本身的不足，引入了 Marionette、Backgrid 和 ModelBinder 等框架和库。

除了以上提到，还引入了一些解决其他开发上问题的第三方库和插件：

  • 基于 HTML5 Forms 的表单验证插件 H5F；
  • Bootstrap 风格的日期时间拾取器 Bootstrap 3 Date/Time Picker；
  • 多选框和单选框美化的 iCheck；
  • 下拉列表美化及功能加强的 Select2
  • 富文本编辑器 Simditor；
  • ……
